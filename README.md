# GuessTheTweet

Set `TOKEN` as a env variable in your .zshrc file or .bashrc

```sh
export TOKEN=Twitter token
```

# Install the dependencies

```s
$ mix deps.get
```

# Run the server

```s
$ mix run --no-halt
```
